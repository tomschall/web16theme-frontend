## Description

Call To Action

## Integration

HTML einbauen.

### Fields

* **img**: Objekt mit Informationen des Bilds
	* **src**: Quellangabe
	* **alt**: Alternativtext
* **text**: Zusätzlicher Beschreibungstext
* **buttonText**: Text der für den Link im Accessible-Modus verwendet werden soll
* **url**: URL zur Seite mit weiteren Informationen
